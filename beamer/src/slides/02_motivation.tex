\subsection{负载监控}

\begin{frame}
\frametitle{研究动机}
\framesubtitle{目标1：负载监控}
容器负载监控机制对云数据中心\textbf{\alert{资源配置和调度决策}}有较大影响

\begin{block}{负载监控机制}
\begin{itemize}
    \item<1-> 基于负载采集程序：负载变化频繁、异构网络环境导致滞后性
    \item<2-> 负载预测模型：主动进行资源配置和容器调度
\end{itemize}
\end{block}

\subsection{容器调度}

\begin{frame}
\frametitle{研究动机}
\framesubtitle{目标2：容器调度}
\begin{itemize}
    \item<1> 任务调度：任务分配 $\Rightarrow$ 缩短任务时间和降低能耗
    \item<2> 虚拟机调度：迁移、放置 $\Rightarrow$ 保证QoS和降低能耗
\end{itemize}
\only<1>{
\begin{block}{容器调度 VS 任务调度}
\begin{itemize}
    \item 生命周期
    \item 状态管理
    \item 动态调整
\end{itemize}
\end{block}
}
\only<2>{
\begin{block}{容器调度 VS 虚拟机调度}
\begin{itemize}
    \item 资源隔离
    \item 迁移开销与延迟
    \item 调度粒度
    \item 依赖距离
\end{itemize}
\end{block}
\bigskip
}
\end{frame}
